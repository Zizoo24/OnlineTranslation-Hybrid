Document Checklist for Translation

Everything you need to prepare before submitting documents for translation. Organized by service type.

Before You Start

  • Ensure all documents are clear and legible - blurry photos may delay processing
  • Send complete documents - all pages including blank ones if present
  • For multi-page documents, send in correct order
  • Let us know if you need hard copies or digital only
  • Mention any deadlines when requesting a quote

Golden Visa Documents

Passport - Bio page with validity
Degree Certificate - Original or attested copy
Degree Equivalency - From MOE if required
Salary Certificate - Current employer, showing AED 30,000+ salary
Employment Contract - Showing job title and salary
Title Deed - If applying via property investment (AED 2M+)
Learn more about Golden Visa Translation

Attestation Documents

Original Document - Not photocopies
Home Country Attestation - Notary, Apostille, or Foreign Ministry
UAE Embassy Stamp - If obtained from home country
Passport Copy - For processing
Learn more about Attestation Services

Family Visa Documents

Marriage Certificate - Attested and translated
Birth Certificates - For children, attested
Sponsor's Passport - UAE resident sponsoring family
Sponsor's Visa Page - Current valid visa
Salary Certificate - Meeting minimum requirements

Corporate/Business Documents

Trade License - Current and valid
Memorandum of Association - MOA
Board Resolutions - If applicable
Power of Attorney - If delegating signing authority
Shareholder Documents - Passport copies of shareholders
Learn more about Corporate Translation
